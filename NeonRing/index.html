Web Ring

What is a webring?

A webring (or web ring) is a collection of websites linked together in a circular structure, and usually organized around a specific theme, often educational or social. They were popular in the 1990s and early 2000s, particularly among amateur websites.

To be a part of the webring, each site has a common navigation bar. It contains links to the previous and next site. By selecting next (or previous) repeatedly, the user will eventually reach the site they started at. This is the origin of the term webring. The select-through route around the ring is usually supplemented by a central site with links to all member sites; this prevents the ring from breaking completely if a member site goes offline. The advantage of a webring is that if the user is interested in the topic on one website, they can quickly connect to another website on the same topic. Webrings usually have a moderator who decides which pages to include in the webring. After approval, webmasters add their pages to the ring by 'linking in' to the ring; this requires adding the necessary HTML or JavaScript widget to their site.

Neon Ring

Neon Ring Manifesto:

Is your website Bright, Flashy, Colorful, or Gaudy? If you think that you fall into any of those categories, you are welcome here in the Neon Ring!.

Sites will not be allowed to participate if they contain any of the following:

  • Bullying, Harassment, Sexism, Homophobia/Transphobia, Racism, or Hate speech.
  • Antisocial / Disturbing behavior
  • Unmarked NSFW content
  • Paid advertisements
  • Malicious code
  • Only a portfolio.

How to join:

1: Let us know that you are interested!

Contact Gray on the discord server and send a message in #web-ring. Alternatively, you can add yourself to the webring by making a pull request on github.

2: Add the webring code to your site.

There are two ways to add the webring to your website.

  1. Add the Javascript script to your site.
  2. Add the HTML anchor tags to your site.

The Javascript <script></script> loads and runs the code from https://graycot.com/webring/loop-ring.js. This creates the widget you see below:


The Javascript widget may take a quarter second to appear when loaded.

The HTML tags direct the user to https://graycot.com/neonring/loop-redirect.html. This page then redirects the user to the previous, member list, home, random site, or next webpage in the ring. You can see an example of the unstyled anchor tags below:

< ... Neon Ring ? >


The HTML widget may delay users up to half a second when being redirected to or from your page.

Copy & paste the JavaScript or HTML into the body of the webpage you want the loop ring to direct visitors.

Note: Things may not work as intended if you include the webring code more than once on a HTML page.

1. JavaScript

<div id="NeonRingJS"><script src="./neon-ring.js" type="module"></script></div>

2. HTML


<a href="./neon-ring-redirect.html?action=prev"> < </a><br />

<a href="./neon-ring-redirect.html?action=list"> ... </a><br />

<a href="./neon-ring-redirect.html?action=home">Neon Ring</a><br />

<a href="./neon-ring-redirect.html?action=rand"> ? </a><br />

<a href="./neon-ring-redirect.html?action=next"> > </a>

You can override the default CSS ruleset by including your own CSS. You must append ".override" to the end of ".neon-ring". Here is how that would look:

.neon-ring.override {} .neon-ring.override a:hover {}

2. HTML

As for the HTML, there is no default styling built in. The links will take on whatever styling you have applied to your webpage. Copy and paste the default JavaScript styling shown above, or design your own style!
